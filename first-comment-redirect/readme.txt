== Description ==

 Redirects commenter to your desired page or custom link who just made their first comment on your site. You can use this page to promote your site by thanking the user for their comment and asking them to subscribe your blog, like your blog on facebook, follow on twitter and many more. Also has the option to redirect every comment. The plugin is also translation ready.

== Installation ==

1. Install the plugin either via the WordPress.org plugin directory, or by uploading the files to your server (in the /wp-content/plugins/ directory).
2. Activate the First Comment Redirect plugin through the 'Plugins' menu in WordPress.
3. Configure the plugin by going to the First Comment Redirect menu that appears in your admin menu.
